2. Data we collect
   - The app does not run analytics or send any personal data to our servers.
   - We have no servers. The app talks directly to: yts.mx public API (movie
     metadata), YouTube (trailer playback in WebView), the BitTorrent peer
     network (for downloads).
   - Local-only data stored on your device: favourites list, search history,
     download state, cached posters, app preferences. Stored in Hive boxes
     inside the app sandbox. Deleted on uninstall.

3. Permissions we request
   - Internet: fetch movie catalog, trailers, torrent peers.
   - Notifications: show download progress.
   - Foreground service, wake lock: keep downloads running.
   - All files access (Android): save downloaded movies to public
     Downloads folder.
